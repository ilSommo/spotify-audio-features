Spotify audio features
======================

The aim of this project is to apply probabilistic reasoning to a Bayesian Network created using audio features of songs on Spotify.

Packages
--------

`python 3.9.4`

`jupyter 1.0.0`  
`matplotlib 3.4.1`  
`numpy 1.20.2`  
`pandas 1.2.4`  
`pgmpy 0.1.14`
